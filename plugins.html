Lulo

A plugin engine for AWS CloudFormation Custom Resources

View project on GitHub

Plugins

All (known) lulo plugins are listed below.

Cognito

  • Cognito Identity Pool Creates Identity Pools for Amazon Cognito
  • Cognito Identity Pool Roles Creates Roles for Identity Pools
  • Cognito User Pools Creates User Pools for Amazon Cognito
  • Cognito User Pool Attributes Creates User Pool Attributes
  • Cognito User Pool Clients Creates User Pool Clients
  • Cognito User Pool Groups Creates User Pool Groups

Elastic Transcoder

  • Elastic Transcoder Pipeline: Creates an Elastic Transcoder Pipeline
  • Elastic Transcoder Preset: Creates an Elastic Transcoder Preset

Other AWS Services

  • ACM Get Certificate Gets the certificate ARN from ACM for a given domain name.
  • Amazon Inspector Role Associates Amazon Inspector with an IAM Role
  • API Gateway Domain Name Creates a domain name for API Gateway
  • EC2 Describe Elastic IP Address: Describes an existing EC2 Elastic IP Address.
  • KMS Grant: Manages AWS KMS Key grants
  • S3 bucket Notifications: Creates bucket notification separate from the Bucket Resource.
  • SES Identity Policy: Creates SES Identity Polices
  • SNS Platform Application: Create an SNS Platform Application
  • Stack-Properties: Yields Parameters, Outputs and Resources given another CloudFormation StackId or StackName

Miscellaneous

  • GUID: Generates a GUID
  • lowercase: Lowercase the input string
  • Multiply: Multiplies two factors and yields the product
  • Sleep: Pause CloudFormation execution for the given number of seconds
  • Substring: Returns a part of the input string

You can also search npm for plugins.
